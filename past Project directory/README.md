# VistaAI 🏡✨

**VistaAI** is an advanced AI-powered pipeline that automatically transforms empty room images into premium, virtually staged video tours. It orchestrates a suite of Generative AI models to analyze, design, animate, and narrate property showcases.

## 🚀 Features

*   ** automated Room Analysis**: Detects room types (Living Room, Kitchen, etc.) using Vision AI.
*   **Virtual Staging**: Generates high-quality furniture and decor overlays while preserving original architecture.
*   **Video Generation**: Creates smooth, cinematic "Before & After" transitions using Google Veo.
*   **Smart Narration**: Writes and synthesizes context-aware voiceovers synced to the visual tour.
*   **Seamless Editing**: Automatically stitches multiple rooms with cross-fade transitions and text overlays.

## 🛠️ Architecture

The pipeline consists of five orchestrated phases:

1.  **Vision & Analysis**: `Gemini 2.0 Flash` analyzes the image mechanics.
2.  **Image Transformation**: `Imagen` (via Gemini) stages the room.
3.  **Video Synthesis**: `Veo` generates the motion transition.
4.  **Narration Engine**: `Gemini 2.0 Flash` writes the script + `Gemini TTS` generates audio.
5.  **Post-Processing Studio**: `FFmpeg` handles text overlays, audio merging, and cross-fade assembly.

## 📦 Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/23f2002284/VistaAI.git
    cd VistaAI
    ```

2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

3.  Set up environment variables:
    Create a `.env` file and add your Google Gemini API Key:
    ```ini
    GEMINI_API_KEY=your_api_key_here
    ```

4.  Ensure **FFmpeg** is installed and added to your system PATH.

## 🏃‍♂️ Usage

Start the API server:
```bash
uvicorn logics.main:app --reload
```

### Endpoints

*   **POST** `/generate-room-tour`: Generate a video for a single image.
*   **POST** `/generate-full-property-tour`: Orchestrate a full multi-room tour.

**Example Request:**
```json
{
  "image_paths": [
    "C:/absolute/path/to/living_room.jpg",
    "C:/absolute/path/to/kitchen.jpg"
  ],
  "preference": "Modern Minimalist",
  "rough_notes": "Emphasize the natural light."
}
```
